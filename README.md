## ✨ Features

### Core Capabilities
- 🔍 **Intelligent ID Guessing** - Multiple strategies for ID enumeration
- 🚀 **Multi-Threaded Scanning** - Fast parallel processing of targets
- 🔌 **Plugin Auto-Installation** - Automatic WP Console deployment
- 💉 **Persistent Shell Deployment** - PHP backdoor installation
- 👤 **Admin Account Creation** - Automatic privileged user creation
- 📊 **Multi-Format Reporting** - JSON, CSV, TXT output formats

### Advanced Features
- 🎯 **WordPress Version Detection** - Automatic CMS version fingerprinting
- 🔄 **Multiple Exploitation Headers** - 9+ different header injection methods
- 🌐 **Proxy Support** - HTTP/HTTPS proxy configuration
- 💾 **Result Backup System** - Automatic backup of previous results

- 📈 **Progress Tracking** - Real-time exploitation status
